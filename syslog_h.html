Behaviours: gen_event.

This module defines the syslog_h behaviour.
Required callback functions: to_iolist/1.

Description

The event handler for to be attached to the error_logger event manager. This module will handle all log message, format them and forward them to the configured protocol handler. Protocol handler must implement the syslog_h behaviour.

attach/1

attach(Socket::gen_udp:socket()) -> ok | term()

Attach this module as event handler for error_logger events. The connection between the event manager and the handler will be supervised by the calling process. The handler will automatically be detached when the calling process exits.
