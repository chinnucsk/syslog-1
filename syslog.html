Main module of the syslog application.

Behaviours: application, supervisor.

This module defines the syslog behaviour.
Required callback functions: to_iolist/1.

Description

Main module of the syslog application. This module contains functions to enable or disable logging via syslog as well as two convinience functions to log messages with a certain severity.

This module also defines the behaviour that must be implemented by protocol backends.
